# DBproject

### 프로젝트 주제
  + 헬스장 회원관리 프로그램

### 프로젝트 설명
  + 회원등록 및 관리를 해주는 프로그램
    + 회원 개인정보 저장
    + 트레이너 정보 저장
    + 회원권 구매
    + PT 이용권 구매
    + 회원 운동 일지 작성
  + 트레이너 정보가 있어 회원들이 트레이너를 선택할 때 도움이 된다.
  + 운동 일지를 작성하여 신체 변화를 파악할 수 있게한다.
